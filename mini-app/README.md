# Кибер-Защитник — Telegram Mini App

Игровой модуль для обучения детей основам безопасности в сети. Курсовая ВШЭ, 1 курс, Программная инженерия 09.03.04. Автор фронта — Никитин-Келлер Ю.Н., бэк — Фролов Б.В.

## Что внутри

- **Каркас Mini App** — точка входа, главное меню 6 игр, прогресс, темизация под Telegram.
- **Сами игры** — отдельные подпроекты в `games/` корневой папки курсовой (`../games/`), пока в виде сидов. Подключаются после готовности.

## Структура

```
mini-app/
├── index.html              # точка входа
├── styles/main.css         # тёмная тема, mobile-first
├── src/
│   ├── main.js             # роутер, бутстрап TG SDK
│   ├── menu.js             # главное меню
│   ├── games.js            # реестр 6 игр (id, name, иконка, описание)
│   ├── progress.js         # localStorage, звания, общий счёт
│   ├── api.js              # заглушка под REST API Фролова
│   ├── theme.js            # маппинг themeParams → CSS-переменные
│   ├── haptics.js          # HapticFeedback + vibrate-фолбэк
│   └── dom.js              # маленький хелпер createElement
├── games/                  # сюда переедут игры по мере готовности
├── questions_frolov.md     # вопросы партнёру — отправить и получить ответы
└── README.md
```

## Запуск локально

ES-модули требуют HTTP-протокол (не `file://`). Самое простое:

```bash
cd "/mnt/c/Users/frosa/Рабочий стол/ВШЭ/Курсовая/mini-app"
python3 -m http.server 8080
```

Открыть в браузере: <http://localhost:8080>

В обычном браузере объект `window.Telegram.WebApp` отсутствует — каркас работает с фолбэками (хаптика через `navigator.vibrate`, тема дефолтная, BackButton недоступен).

## Тест внутри Telegram

1. Прокинуть локальный сервер наружу: `npx localtunnel --port 8080` (или `ngrok http 8080`).
2. У `@BotFather`: `/newapp` → выбрать бота Фролова → указать публичный HTTPS URL.
3. Открыть бота, нажать кнопку Mini App.

## Соглашения

- **Тач-таргеты** ≥ 44px. Никакой клавиатуры (Mini App открывается на мобилке вертикально).
- **HapticFeedback** на ключевые события: выбор плитки, победа, ошибка.
- **Хранение прогресса** локально (localStorage); API Фролова — оверлей сверху.
- **Тёмная тема** через `themeParams`, без захардкоженных цветов в JS.
- **Сессия игры** 1–3 минуты, чтобы заходило в формат коротких заходов.
